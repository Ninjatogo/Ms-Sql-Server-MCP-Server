# Ms-Sql-Server-MCP-Server

This project is a web API for interacting with a Microsoft SQL Server database. It is built with ASP.NET Core and runs on .NET 7.

## Getting Started

To get started, clone this repository to your local machine.

```
git clone <repository-url>
```

### Prerequisites

* [.NET 7 SDK](https://dotnet.microsoft.com/download/dotnet/7.0)
* [Visual Studio 2022](https://visualstudio.microsoft.com/vs/) or another code editor of your choice
* A running instance of Microsoft SQL Server.

### Building and Running the Application

1.  Open the solution file `MsSqlServerMcpServer.sln` in Visual Studio.
2.  Configure the database connection string in `appsettings.json`.
3.  Build the solution by pressing `Ctrl+Shift+B`.
4.  Run the application by pressing `F5`.

The application will start and listen on `https://localhost:5001`.
